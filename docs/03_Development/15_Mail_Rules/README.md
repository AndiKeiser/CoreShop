# CoreShop Mail Rules

Mail Rules are responsible for all types of email correspondence. It handles mails for following types:

 - order
 - invoice
 - shipment
 - user
 - messaging
 - payment

It is designed to add new types. Read more about this [here](./03_Custom_Types.md)
